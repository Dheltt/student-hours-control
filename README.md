# Student Hours Control System

## Overview

The Student Hours Control System is a desktop-based application designed to manage and track activity hours performed by student tax advisors.

The system ensures structured data management, hour tracking, and report generation while maintaining clean architecture principles and scalability.

## Objectives

- Register students
- Register activities
- Track accumulated hours
- Generate reports
- Maintain data integrity

## Tech Stack

- Python 3.12
- MySQL 8
- SQLAlchemy
- Docker & Docker Compose

## Architecture

The system follows a layered architecture:

- Presentation Layer (CLI)
- Service Layer (Business Logic)
- Repository Layer (Data Access)
- Database Layer (MySQL)
